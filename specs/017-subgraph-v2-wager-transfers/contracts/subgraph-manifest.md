# Contract: Subgraph Manifest + Networks (`subgraph/subgraph.yaml` + `subgraph/networks.json`)

Replaces the legacy `FriendGroupMarketFactory` data source with the v2 `WagerRegistry`.
Address + startBlock come from `networks.json` (built per network via `graph build --network`).

## Data source (single manifest)

```yaml
specVersion: 0.0.5
description: v2 WagerRegistry wagers + per-transfer records for the prediction-dao-research app
schema:
  file: ./schema.graphql
dataSources:
  - kind: ethereum
    name: WagerRegistry
    network: polygon              # overridden per build by --network / networks.json
    source:
      address: "0x5023765809fDA93ab9F11B684fdb76521eD31774"   # from networks.json
      abi: WagerRegistry
      startBlock: 88118344                                     # from networks.json (NEVER 0)
    mapping:
      kind: ethereum/events
      apiVersion: 0.0.7
      language: wasm/assemblyscript
      entities:
        - Wager
        - WagerTransfer
      abis:
        - name: WagerRegistry
          file: ../frontend/src/abis/WagerRegistry.json   # GENERATED by sync:frontend-contracts
      eventHandlers:
        - event: WagerCreated(indexed uint256,indexed address,indexed address,address,uint128,uint128,uint8,bytes32,string)
          handler: handleWagerCreated
        - event: WagerAccepted(indexed uint256,indexed address)
          handler: handleWagerAccepted
        - event: PayoutClaimed(indexed uint256,indexed address,uint256)
          handler: handlePayoutClaimed
        - event: WagerRefunded(indexed uint256,indexed address,indexed address)
          handler: handleWagerRefunded
        - event: WagerDrawn(indexed uint256,indexed address,indexed address,address)
          handler: handleWagerDrawn
        - event: WagerCancelled(indexed uint256)
          handler: handleWagerCancelled
        - event: WagerResolved(indexed uint256,indexed address,indexed address)
          handler: handleWagerResolved
        # status completeness (optional, no transfer rows):
        - event: WagerDeclined(indexed uint256,indexed address)
          handler: handleWagerDeclined
        - event: DrawProposed(indexed uint256,indexed address)
          handler: handleDrawProposed
        - event: DrawRevoked(indexed uint256,indexed address)
          handler: handleDrawRevoked
      file: ./src/mappings/wagerRegistry.ts
```

Event signatures are verified against `frontend/src/abis/WagerRegistry.js`.

Sourced from `deployments/<net>-v2.json` (extended to record `wagerRegistry` deploy block).
Polygon's block (88118344) is already known in `frontend/src/config/contracts.js`.

## Acceptance hooks

- No data source may carry `address: "0x0…0"` or `startBlock: 0` (SC-005).
- The `abis[].file` MUST be the generated `WagerRegistry.json`, not a hand-edited copy.
- One deployment per network; its endpoint becomes that network's `VITE_SUBGRAPH_URL`.
